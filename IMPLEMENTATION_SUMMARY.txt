================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "Python is not installed or not in PATH"
  SOLUTION: Install Python from https://www.python.org/
            Make sure to check "Add Python to PATH" during installation

PROBLEM: "yt-dlp is not installed"
  SOLUTION: Run: pip install yt-dlp
            Or: pip install -r requirements.txt

PROBLEM: "No video formats available"
  SOLUTION: - Video might be private or age-restricted
            - Try a different video to verify the app works
            - Some videos have download restrictions

PROBLEM: Download fails with network error
  SOLUTION: - Check your internet connection
            - Wait a few seconds and try again
            - Try a different video

PROBLEM: FFmpeg error when downloading MP3
  SOLUTION: Run: pip install yt-dlp[default]
            This installs yt-dlp with FFmpeg included

PROBLEM: Application window doesn't appear
  SOLUTION: - Run from Command Prompt to see error messages
            - Make sure Tkinter is available (python -m tkinter)
            - Check Windows firewall settings
